Cloud Services

How to export and repair your photo libraries from the cloud.

Key Export Strategies

  • Google Photos: Use Google Takeout. Always ensure JSON sidecar files are included in your download. These contain the original dates and metadata that PDR uses to restore your timeline.
  • iCloud / Apple Photos: Select "Export Unmodified Original" when exporting from the Mac Photos app. This preserves the original file creation dates and metadata.
  • OneDrive / Dropbox: Simply download your folders. PDR can scan through nested folder structures to find and fix dates.
  • Social Media: For platforms like Facebook or Instagram, request a full data download to get the highest quality images available, even if metadata is stripped.

Hardware Devices

Strategies for old phones, cameras, SD cards, and hard drives.

Recovering from Hardware

  • Phones & Cameras: Connect your device via USB and copy the entire `DCIM` folder. This usually contains the most original version of your photos with intact metadata.
  • Old Devices: For devices that no longer turn on, check if they have a removable SD card. Use a card reader to extract the files.
  • External Drives: If you have "backup" folders scattered across multiple hard drives, consolidate them into one place. PDR will identify duplicates and fix dates across the merged collection.
  • Corrupted Systems: If a file system crash messed up your file modification dates, PDR relies on internal file headers (EXIF) to restore the correct timeline.
